Introduction
============

An installable theme for Plone 3.0 by CANAIMA GNU/Linux Website

Based on the Canaima Aponwao theme Luis Alejandro Martínez Faneyth 
(http://huntingbears.co.cc/).

This theme has not gone through the rounds of browser testing
so it should be considered an alpha release.

Luis Alejandro Martínez Faneyth was based on a WordPress theme 
called FREEmium released for free by FreebiesDock.com
Authors: Paul Kadysz (design) and Dariusz Siedlecki (frontend)
